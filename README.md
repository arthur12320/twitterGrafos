to use it, create a file in the root of the project 
called: yourData.js

inside that file put:

```javascript
module.exports = {  
    "consumerKey": "XXX",  
    "consumerSecret": "XXX",   
    "accessToken": "XXX",  
    "accessTokenSecret": "XXX",  
    "callBackUrl": "XXX"  
}
```

remember to run:

```console
foo@bar:~$ npm i 
```
